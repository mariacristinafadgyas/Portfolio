My projects

My Zootopia

The core functionality of this project is built with Python

Movie App V3.0

Movie App 3 builds on previous versions by introducing a robust OOP approach that improves the modularity and maintainability of the code. This version includes new classes for the Movie App and an interface that abstracts storage options to provide more flexibility and scalability. The project continues to provide seamless movie management that integrates API-based information retrieval and improved graphical navigation.

Nose Savers

This app provides timely SMS alerts to subscribers about pollen levels in their specified location, helping them stay informed and manage allergies effectively. Users receive messages containing the time of the maximum pollen quantity for their selected trees, ensuring they have precise information to plan their day.

Rock Paper Scissors

Rock Paper Scissors is a simple but classic hand game in which each player simultaneously forms one of three shapes: Rock, Paper or Scissors. This project uses an object-oriented approach with a Player class as a base that supports different player types such as Player (always plays "rock"), HumanPlayer (user input), RandomPlayer (random move) and CyclePlayer (cyclic move).

Best Buy

The BestBuy app is a Python-based application that uses an object-orientated approach to manage products and facilitate purchases. It includes a product class to display the attributes of each product and a store class to manage a list of products and process transactions. The app allows users to add and remove products, check product availability and process orders efficiently.

SciQuest Journal

SciQuest Journal is a simple web application built with Flask that allows users to create, view, update and delete blog posts. Key features include the ability to create new posts, view all existing posts and update or delete posts using the appropriate buttons; the Like function is also implemented. The application displays the title, author, content and number of likes for each post.

Blog API

A RESTful API built with Python and Flask for managing blog posts. It supports CRUD operations, advanced search, and JSON data storage. Designed for scalability and ease of use, it's ideal for content management systems or websites. API documentation is provided using Swagger for seamless integration and development.

Voice AI Chatbot

A voice-enabled AI chatbot that lets users ask questions through text or speech and receive humorous, AI-generated responses. The chatbot provides voice feedback with animated visual effects and a dynamic, interactive UI featuring colorful animations and a "Listening..." indicator for an engaging user experience.

SkySQL

SkySQL is a platform for analyzing and visualizing flight data, featuring a user-friendly web interface and a Flask API. Users can view flights by ID or date, analyze delays by airline or airport, and explore delay patterns with interactive heatmaps and maps. It offers comprehensive tools for tracking flight performance trends, including data visualization and seamless API querying.

My Library

My Library is a web-based Book Library Management System built with Flask and SQLAlchemy, allowing users to manage books and authors by adding, deleting, and viewing details. It features search and sort options for books and authors, user interface customization through a color picker, and book suggestions based on a randomly selected book. Additionally, users can view purchase links for books fetched via the Google Books API, with icons indicating availability.
