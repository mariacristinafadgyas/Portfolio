My Zootopia

This project is designed to fetch detailed information about various animals using the free API provided by api-ninjas.com. By entering the name of an animal, users can retrieve comprehensive data about their chosen animal, which is then dynamically displayed on a web page.
The core functionality of this project is built with Python

Movie Project 1

First version of the Movie Project, a Python-based application for managing films. This programme allows users to seamlessly add, update, delete and perform fuzzy searches for movies via the command prompt. With bright console colours that enhance user interaction, users can organise their movie collection effortlessly and easily.
The core functionality of this project is built with Python

Movie Project 2

This is the second version of the Movie Project. This version uses an API to seamlessly retrieve detailed film information. By integrating an HTML template, users can now explore and retrieve film data with improved graphics and easy navigation.
The core functionality of this project is built with Python

Nose Savers

This app provides timely SMS alerts to subscribers about pollen levels in their specified location, helping them stay informed and manage allergies effectively. Users receive messages containing the time of the maximum pollen quantity for their selected trees, ensuring they have precise information to plan their day.
